📘 Projeto React - Atividade com Rotas e Componentes
Este projeto foi desenvolvido como parte de uma atividade prática de React. Ele contém uma página principal com links para três funcionalidades diferentes, cada uma representando uma rota/página específica.

🏠 Página Principal (Home)
A tela inicial apresenta três botões que redirecionam para as seguintes páginas:

🔘 Questão 1 - Tema Claro/Escuro

🔘 Questão 2 - Mensagem de Boas-Vindas

🔘 Questão 3 - Lista de Tarefas

✅ Questões
1 – Alterar Tema (Claro/Escuro)
Um botão alterna entre os temas claro e escuro.

O texto do botão muda de acordo com o tema atual.

O fundo da tela também é alterado.

2 – Mudar Texto com Botão
Exibe um parágrafo com o texto inicial: "Olá!".

Ao clicar no botão Entrar, o texto muda para: "Bem-vindo!".

3 – Lista de Tarefas
Um campo de input permite digitar tarefas.

Um botão Adicionar insere a tarefa na lista.

As tarefas digitadas são exibidas abaixo em uma lista.
